# Javascript-self-reference
Self reference to example of useful functions of JS.

## array.map
```
let numbers = [1, 4, 9]
let doubles = numbers.map(function(num) {
  return num * 2
})
// doubles é agora [2, 8, 18]. numbers ainda é [1, 4, 9]
```

## array.reverse

```
let numbers = [1, 4]
let revert = numbers.reverse()
// revert é agora [4, 1]
```
